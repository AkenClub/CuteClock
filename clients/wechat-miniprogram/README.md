# 📱 微信小程序客户端

CuteClock 微信小程序版本，提供便捷的移动端设备控制。

## ✨ 功能特性

- 🏠 **设备控制**: 房间灯光开关控制
- 🔆 **亮度调节**: 时钟显示亮度调节
- 🖥️ **电脑控制**: 远程电脑开关机
- 📊 **状态监控**: 实时查看设备状态
- 🌡️ **温度显示**: 室内温度监测（TODO: 未实现）

## 🛠️ 技术栈

- **开发框架**: 微信小程序原生开发
- **UI 组件**: 微信官方组件库
- **网络请求**: wx.request API
- **数据管理**: 本地存储 + 实时获取

## 📁 项目结构

已包含的文件：

- `app.js` - 小程序主入口文件
- `app.json` - 小程序配置文件
- `app.wxss` - 全局样式文件
- `pages/` - 页面文件目录
  - `index/` - 主页面（设备控制）
  - `device-manager/` - 设备管理页面
- `utils/` - 工具函数
- `other/` - 业务逻辑工具类
- `http/` - 网络请求封装
- `project.config.json` - 项目配置

## 🚀 快速开始

### 📋 使用前准备

1. **硬件准备**: 确保您已经搭建好 CuteClock 硬件设备
2. **网络配置**: 设备已连接到与手机相同的 WiFi 网络
3. **微信开发者工具**: 安装微信开发者工具用于调试

### 🔧 部署步骤

1. **导入项目**

   ```bash
   # 使用微信开发者工具打开当前目录
   ```

2. **配置设备信息**

   - 如有必要，更新 API 密钥配置

3. **编译运行**
   - 安装依赖：`npm install`
   - 在微信开发者工具 - 顶部菜单“工具” - “构建 npm”，完成后即可编译项目
   - 使用真机调试或发布体验版

### 📱 功能使用

- **设备发现**: 应用会自动搜索局域网内的设备
- **灯光控制**: 点击灯光按钮开关房间灯具
- **亮度调节**: 滑动调节时钟显示亮度
- **电脑控制**: 远程开关电脑电源
